# Commands
```Shell
apt update && apt upgrade -y
apt install ruby -y
gem insatll lolcat
apt install git
git clone https://github.com/InferiorAK/TermuxStyler
cd TermuxStyler
chmod +x customize.sh
./customize.sh

```
